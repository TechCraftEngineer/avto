# Улучшения раздела вакансий

## 🎯 Цель
Сделать интерфейс вакансий понятным и полезным для рекрутеров и их руководителей, добавив важные метрики и подсказки.

## ✨ Что улучшено

### 1. **Таблица вакансий** (`vacancy-table.tsx`, `vacancy-table-row.tsx`)

#### Изменённые колонки:
- ✅ **"Название вакансии"** вместо "Название" — понятнее для рекрутеров
- ✅ **"Площадка"** вместо "Источник" — более привычный термин
- ✅ **"Всего откликов"** вместо "Отклики" — точнее отражает суть
- ✅ **"Новые"** — отдельная колонка для новых откликов с зелёным бейджем
- ✅ **"В обработке"** вместо "В работе" — профессиональный термин
- ✅ **"Закрыта"** вместо "Неактивна" — понятнее статус

#### Новые возможности:
- 🎨 **Визуальные индикаторы приоритета** — вакансии с новыми откликами подсвечиваются зелёным фоном
- 📊 **Конверсия просмотров в отклики** — показывается под количеством откликов
- 🎯 **Индикатор эффективности** — цветной бейдж с процентом конверсии (зелёный/жёлтый/красный)
- 💡 **Пульсирующая точка** — на вакансиях с новыми откликами
- 📈 **Детализация "В обработке"** — показывает "X из Y" для контекста

### 2. **Новый компонент: Индикатор эффективности** (`vacancy-performance-badge.tsx`)

Показывает качество вакансии по конверсии:
- 🟢 **Отличная** (5%+) — зелёный бейдж с стрелкой вверх
- 🟡 **Средняя** (2-5%) — жёлтый бейдж
- 🔴 **Низкая** (<2%) — красный бейдж со стрелкой вниз

**Польза для рекрутера:** Сразу видно, какие вакансии работают хорошо, а какие нужно улучшить.

### 3. **Новый компонент: Умные подсказки** (`vacancy-insights.tsx`)

Автоматически анализирует данные и даёт рекомендации:

#### Примеры подсказок:
- ⚠️ **"Требуется внимание"** — много новых откликов (>10)
  - *"У вас 15 новых откликов. Рекомендуем обработать их в течение 24 часов"*

- 💡 **"Низкая активность"** — мало откликов на вакансию (<2)
  - *"Среднее количество откликов ниже нормы. Попробуйте улучшить описание"*

- 🎉 **"Отличная работа!"** — высокая активность (10+ откликов)
  - *"В среднем 12 откликов на вакансию — отличный результат!"*

- ℹ️ **"Нет активных вакансий"** — все закрыты
  - *"Создайте новые или активируйте существующие"*

**Польза:** Рекрутер сразу понимает, что требует внимания и как улучшить результаты.

### 4. **Улучшенная статистика** (`vacancy-stats.tsx`)

#### Новые карточки:
- 📊 **"Отклики на вакансию"** — среднее с оценкой качества
  - Показывает: "Отлично" / "Хорошо" / "Средне" / "Низкая"
  
- ✅ **"Закрытые вакансии"** — с процентом успеха
  - Мотивирует и показывает эффективность работы

- ⏱️ **"Время закрытия"** — среднее время в днях
  - Оценка: "Быстро" / "Норма" / "Долго" / "Очень долго"

- 💼 **"Конверсия в найм"** — процент успешных наймов
  - Ключевая метрика для руководителя

**Польза:** Руководитель видит полную картину эффективности команды.

### 5. **Компонент справки** (`vacancy-help-tooltip.tsx`)

Кнопка "?" рядом с каждой метрикой открывает понятное объяснение:

#### Пример для "Всего откликов":
```
📌 Всего откликов
Общее количество кандидатов, откликнувшихся на вакансию 
с момента публикации.

✅ 10+ откликов — вакансия привлекает внимание
⚠️ Менее 2 откликов — стоит улучшить описание или условия
```

**Польза:** Новые рекрутеры быстро понимают, что означает каждая цифра.

### 6. **Улучшенные скелетоны загрузки**

Более реалистичные заглушки при загрузке:
- Показывают структуру данных (название + бейдж, отклики + конверсия)
- Помогают понять, что будет загружено

## 📊 Ключевые метрики для рекрутера

### Приоритет 1: Новые отклики
- Зелёный бейдж с числом
- Пульсирующая точка на вакансии
- Подсветка строки зелёным фоном

### Приоритет 2: Эффективность вакансии
- Цветной бейдж с конверсией
- Процент под количеством откликов
- Подсказка с рекомендациями

### Приорит 3: Воронка найма
- "В обработке" показывает прогресс
- Статистика закрытых вакансий
- Время до закрытия

## 🎨 Визуальные улучшения

1. **Цветовая кодировка:**
   - 🟢 Зелёный — хорошо, требует внимания
   - 🟡 Жёлтый — средне, можно улучшить
   - 🔴 Красный — плохо, нужны действия
   - 🔵 Синий — информация, норма

2. **Анимации:**
   - Пульсация на новых откликах
   - Плавное появление бейджей
   - Hover-эффекты для интерактивности

3. **Адаптивность:**
   - На мобильных скрыты второстепенные колонки
   - Важная информация всегда видна
   - Компактное отображение на планшетах

## 💼 Польза для руководителя

### Быстрая оценка команды:
- Сколько вакансий в работе
- Средняя эффективность (отклики/вакансия)
- Скорость закрытия вакансий
- Конверсия в найм

### Выявление проблем:
- Вакансии без откликов
- Долгие вакансии (>60 дней)
- Низкая конверсия (<2%)
- Необработанные отклики

### Мотивация команды:
- Показатели успеха (закрытые вакансии)
- Позитивные подсказки при хороших результатах
- Прозрачность метрик

## 🚀 Как это работает

1. **Рекрутер открывает раздел вакансий**
   - Видит карточки со статистикой
   - Получает умные подсказки о том, что требует внимания

2. **Смотрит на таблицу**
   - Вакансии с новыми откликами выделены
   - Видит эффективность каждой вакансии
   - Понимает, где нужно улучшить описание

3. **Наводит на "?"**
   - Читает понятное объяснение метрики
   - Узнаёт, какие значения хорошие, а какие плохие

4. **Принимает решения**
   - Обрабатывает новые отклики в первую очередь
   - Улучшает вакансии с низкой конверсией
   - Фокусируется на эффективных каналах
