# Фотографии кандидатов

Эта папка содержит локальные фотографии кандидатов для загрузки в систему.

## Подготовка фотографий

1. Скачайте фотографии для каждого кандидата
2. Переименуйте файлы в соответствии с `candidateId` из `candidate-photos.json`
3. Поддерживаемые форматы: `.jpg`, `.jpeg`, `.png`, `.webp`

## Примеры имен файлов

```
hh_candidate_001.jpg
hh_candidate_002.png
hh_candidate_003.jpg
sj_candidate_001.jpg
...
```

```bash

```
