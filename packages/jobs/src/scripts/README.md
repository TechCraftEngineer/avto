# Скрипты для исправления данных

## fix-vacancy-publications.ts

Исправляет отсутствующие записи в таблице `vacancy_publications` для вакансий, импортированных с HH.ru и других платформ.

### Проблема

При импорте архивных вакансий создавались записи в таблице `vacancy`, но не создавались соответствующие записи в `vacancy_publications`, которые связывают вакансию с платформой публикации.

### Решение

Скрипт находит все вакансии без публикаций и создаёт для них записи в `vacancy_publications` на основе данных из таблицы `vacancy` (source, externalId, url).

### Запуск

```bash
cd packages/jobs
bun run fix:publications
```

### Что делает скрипт

1. Находит все вакансии без записей в `vacancy_publications`
2. Для каждой вакансии создаёт запись публикации с данными:
   - `vacancyId` - ID вакансии
   - `platform` - платформа из поля `source`
   - `externalId` - внешний ID вакансии
   - `url` - ссылка на вакансию
   - `isActive` - статус активности

### Результат

После выполнения скрипта все вакансии будут иметь соответствующие записи в `vacancy_publications`, что позволит корректно отображать информацию о публикациях и связывать отклики с публикациями.
