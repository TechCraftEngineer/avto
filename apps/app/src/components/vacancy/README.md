# Vacancy Domain

Компоненты и логика для работы с отдельными вакансиями.

## Структура

```
vacancy/
├── components/          # React компоненты (10 компонентов)
├── hooks/               # Кастомные React хуки
├── utils/               # Утилитарные функции
├── types/               # TypeScript типы
└── index.ts             # Экспорты домена
```

## Компоненты

Домен содержит 10 компонентов для работы с отдельными вакансиями.

## Использование

```typescript
import { VacancyEditor } from '@/components/vacancy';
```

## Архитектура

Домен следует принципам:
- **Доменная архитектура** - компоненты сгруппированы по бизнес-логике
- **Прямые импорты** - оптимизация bundle size
- **Server Components** - по умолчанию для производительности
- **TypeScript** - строгая типизация
