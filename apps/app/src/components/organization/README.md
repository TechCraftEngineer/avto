# Organization Domain

Компоненты и логика для работы с организациями и командами.

## Структура

```
organization/
├── components/          # React компоненты (8 компонентов)
└── index.ts             # Экспорты домена
```

## Компоненты

Домен содержит 8 компонентов для работы с организациями и командами.

## Использование

```typescript
import { CreateOrganizationDialog } from '@/components/organization';
```

## Архитектура

Домен следует принципам:
- **Доменная архитектура** - компоненты сгруппированы по бизнес-логике
- **Прямые импорты** - оптимизация bundle size
- **Server Components** - по умолчанию для производительности
- **TypeScript** - строгая типизация
