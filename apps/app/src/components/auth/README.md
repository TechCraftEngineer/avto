# Auth Domain

Компоненты и логика для работы с аутентификацией и авторизацией пользователей.

## Структура

```
auth/
├── components/          # React компоненты (11 компонентов)
├── hooks/               # Кастомные React хуки
├── utils/               # Утилитарные функции
├── types/               # TypeScript типы
└── index.ts             # Экспорты домена
```

## Компоненты

Домен содержит 11 компонентов для работы с аутентификацией и авторизацией пользователей.

## Использование

```typescript
import { LoginForm } from '@/components/auth';
```

## Архитектура

Домен следует принципам:
- **Доменная архитектура** - компоненты сгруппированы по бизнес-логике
- **Прямые импорты** - оптимизация bundle size
- **Server Components** - по умолчанию для производительности
- **TypeScript** - строгая типизация
