# Getting-started Domain

Компоненты и логика для работы с введением новых пользователей.

## Структура

```
getting-started/
├── components/          # React компоненты (3 компонентов)
└── index.ts             # Экспорты домена
```

## Компоненты

Домен содержит 3 компонентов для работы с введением новых пользователей.

## Использование

```typescript
import { GettingStartedWidget } from '@/components/getting-started';
```

## Архитектура

Домен следует принципам:
- **Доменная архитектура** - компоненты сгруппированы по бизнес-логике
- **Прямые импорты** - оптимизация bundle size
- **Server Components** - по умолчанию для производительности
- **TypeScript** - строгая типизация
