# Response-detail Domain

Компоненты и логика для работы с деталями откликов.

## Структура

```
response-detail/
├── components/          # React компоненты (4 компонентов)
├── hooks/               # Кастомные React хуки
├── utils/               # Утилитарные функции
└── index.ts             # Экспорты домена
```

## Компоненты

Домен содержит 4 компонентов для работы с деталями откликов.

## Использование

```typescript
import { ResponseDetailCard } from '@/components/response-detail';
```

## Архитектура

Домен следует принципам:
- **Доменная архитектура** - компоненты сгруппированы по бизнес-логике
- **Прямые импорты** - оптимизация bundle size
- **Server Components** - по умолчанию для производительности
- **TypeScript** - строгая типизация
