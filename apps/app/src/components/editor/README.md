# Editor Domain

Компоненты и логика для работы с текстовым редактором.

## Структура

```
editor/
├── components/          # React компоненты (1 компонентов)
└── index.ts             # Экспорты домена
```

## Компоненты

Домен содержит 1 компонентов для работы с текстовым редактором.

## Использование

```typescript
import { TipTapEditor } from '@/components/editor';
```

## Архитектура

Домен следует принципам:
- **Доменная архитектура** - компоненты сгруппированы по бизнес-логике
- **Прямые импорты** - оптимизация bundle size
- **Server Components** - по умолчанию для производительности
- **TypeScript** - строгая типизация
