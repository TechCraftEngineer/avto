# Gigs Domain

Компоненты и логика для работы с gigs - платными проектами для фрилансеров.

## Структура

```
gigs/
├── components/          # React компоненты (27 компонентов)
├── hooks/               # Кастомные React хуки
├── utils/               # Утилитарные функции
├── types/               # TypeScript типы
└── index.ts             # Экспорты домена
```

## Компоненты

Домен содержит 27 компонентов для работы с gigs:
- Карточки и списки gigs
- Формы создания и редактирования
- Компоненты сравнения кандидатов
- Фильтры и поиск
- Детальные представления

## Использование

```typescript
import { GigCard, GigsList } from '@/components/gigs';
```

## Архитектура

Домен следует принципам:
- **Доменная архитектура** - компоненты сгруппированы по бизнес-логике
- **Прямые импорты** - оптимизация bundle size
- **Server Components** - по умолчанию для производительности
- **TypeScript** - строгая типизация
