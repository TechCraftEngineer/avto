# Workspace Domain

Компоненты и логика для работы с рабочим пространством.

## Структура

```
workspace/
├── components/          # React компоненты (9 компонентов)
└── index.ts             # Экспорты домена
```

## Компоненты

Домен содержит 9 компонентов для работы с рабочим пространством.

## Использование

```typescript
import { WorkspaceSwitcher } from '@/components/workspace';
```

## Архитектура

Домен следует принципам:
- **Доменная архитектура** - компоненты сгруппированы по бизнес-логике
- **Прямые импорты** - оптимизация bundle size
- **Server Components** - по умолчанию для производительности
- **TypeScript** - строгая типизация
