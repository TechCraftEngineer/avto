# Chat Domain

Компоненты и логика для работы с чатом и коммуникациями.

## Структура

```
chat/
├── components/          # React компоненты (33 компонентов)
└── index.ts             # Экспорты домена
```

## Компоненты

Домен содержит 33 компонентов для работы с чатом и коммуникациями.

## Использование

```typescript
import { ChatPanel } from '@/components/chat';
```

## Архитектура

Домен следует принципам:
- **Доменная архитектура** - компоненты сгруппированы по бизнес-логике
- **Прямые импорты** - оптимизация bundle size
- **Server Components** - по умолчанию для производительности
- **TypeScript** - строгая типизация
