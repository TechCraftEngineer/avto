# Архивированный импорт вакансий

Компоненты импорта вакансий через веб-интерфейс (активные, архивные, по ссылке).

Сейчас для импорта вакансий рекомендуется использовать расширение **«Помощник рекрутера»** — оно работает на страницах HeadHunter и позволяет выбирать вакансии прямо на сайте.

Код сохранён на случай, если понадобится вернуть импорт через веб-интерфейс.
