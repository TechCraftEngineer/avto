# Onboarding Domain

Компоненты и логика для работы с регистрацией пользователей.

## Структура

```
onboarding/
├── components/          # React компоненты (6 компонентов)
└── index.ts             # Экспорты домена
```

## Компоненты

Домен содержит 6 компонентов для работы с регистрацией пользователей.

## Использование

```typescript
import { OrganizationForm } from '@/components/onboarding';
```

## Архитектура

Домен следует принципам:
- **Доменная архитектура** - компоненты сгруппированы по бизнес-логике
- **Прямые импорты** - оптимизация bundle size
- **Server Components** - по умолчанию для производительности
- **TypeScript** - строгая типизация
