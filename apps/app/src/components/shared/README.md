# Shared Components

Переиспользуемые компоненты, которые используются в нескольких доменах приложения.

## Структура

```
shared/
├── components/          # React компоненты (22 компонента)
├── utils/               # Утилитарные функции
├── types/               # TypeScript типы
└── index.ts             # Экспорты домена
```

## Компоненты

Домен содержит 22 компонента для общих нужд:
- Карточки кандидатов и профилей
- Детальные карточки откликов
- Табы и секции для детального просмотра
- Общие UI элементы

## Основные компоненты

- `ParsedProfileCard` - карточка распарсенного профиля кандидата
- `ResponseDetailCards` - карточки деталей отклика
- `ResponseDetailTabs` - табы для детального просмотра отклика

## Использование

```typescript
import { ParsedProfileCard } from '@/components/shared';
```

## Архитектура

Домен следует принципам:
- **Переиспользуемость** - компоненты для множественного использования
- **Прямые импорты** - оптимизация bundle size
- **Server Components** - по умолчанию для производительности
- **TypeScript** - строгая типизация
