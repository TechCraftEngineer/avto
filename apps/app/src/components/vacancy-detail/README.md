# Vacancy-detail Domain

Компоненты и логика для работы с деталями вакансий.

## Структура

```
vacancy-detail/
├── components/          # React компоненты (0 компонентов)
├── utils/               # Утилитарные функции
└── index.ts             # Экспорты домена
```

## Компоненты

Домен содержит утилиты и типы для работы с деталями вакансий.

## Использование

```typescript
import { SourceConfig } from '@/components/vacancy-detail';
```

## Архитектура

Домен следует принципам:
- **Доменная архитектура** - компоненты сгруппированы по бизнес-логике
- **Прямые импорты** - оптимизация bundle size
- **Server Components** - по умолчанию для производительности
- **TypeScript** - строгая типизация
