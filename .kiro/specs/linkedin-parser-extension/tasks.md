# План реализации: Recruitment Assistant Extension

## Обзор

Этот план описывает пошаговую реализацию Chrome-расширения для автоматизации работы рекрутеров с профилями кандидатов на различных платформах (LinkedIn, HeadHunter). Расширение использует расширяемую архитектуру с адаптерами для каждой платформы.
